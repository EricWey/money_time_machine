# 钱值时光机 产品需求文档（PRD）
Version：v1.2
Date：2026/04

## 项目背景与目标
在中国宏观经济快速发展背景下，名义货币价值（面值）与实际购买力存在巨大鸿沟，用户对“钱变少了”感知强烈，但缺乏科学量化工具。

- 一句话定义：基于宏观经济数据与AI语义理解的**货币购买力时空坐标系**
- 核心指标：数据真实感（专业）、文案共鸣度（有趣）、分享回流率（裂变）
- 专业工具：基于CPI、M2及房价权重的权威货币换算逻辑
- 情感链接：通过怀旧实物对比，引发社交共鸣
- 决策辅助：为用户提供直观的资产漂流建议

## 核心功能详述
### 功能模块一：时空等值转换器
- 业务逻辑：解决“名义价格”与“实际价值”脱节，调用**三位一体计算模型**
- 输入规范：
  - 金额（Amount）：1-99,999,999（整数）
  - 起始年份（Source_Year）：1980-2026（下拉选择）
  - 城市（Location）：全国地级市（手动选择）
- 输出规范：
  - 等值金额：算法修正后的现值
  - 实体矩阵：3-5个维度实物对比（猪肉、房价、大米等）

### 功能模块二：财富漂流瓶
- 业务逻辑：采用**一篮子商品加权模型**进行跨地域测算
- 输入规范：
  - 当前所在城市：全国地级市（必填）
  - 当前月收入：1-99,999,999（整数，必填）
  - 总资产：1-99,999,999（整数，可选）
  - 目标城市：全国地级市（必填）
- 输出规范：
  - 阶层判定：输出身份标签（中产、财富自由、赤贫等）
  - 购买力倍数：目标城市相对所在城市的购买力系数（COLI比值）

### 功能模块三：AI深度文案生成（RAG驱动）
- 业务逻辑：后端调用LLM结合向量数据库
- 功能：根据计算结果自动生成50字内“扎心”或“暖心”文案，支持一键生成海报
- LLM提示词：XXXXX

## 核心算法与数据逻辑
### 三位一体货币价值模型
Value_now = Value_past × (0.4·CPI_now/CPI_past + 0.4·M2_adj_now/M2_adj_past + 0.2·Property_now/Property_past)


1. **锚点1：CPI（权重40%）**
   - 定义：反映基础生存成本（食品、服装、水电）
   - 来源：国家统计局年度CPI环比累计值
2. **锚点2：M2_adj（权重40%）**
   - 定义：M2增速 - GDP增速
   - 逻辑：捕捉超发货币流入非消费领域导致的资产稀释，非民生领域通胀
3. **锚点3：Property（权重20%）**
   - 定义：目标年份与当前年份当地平均房价比值
   - 逻辑：房产在中国具有强货币锚定属性

### 城市生活成本指数（COLI）
针对“财富漂流”功能，建立跨城市、跨年代评价体系，对比六大维度：

| 维度 | 权重 | 代表性实物（MVP采集项） |
| ---- | ---- | ---------------------- |
| 居住（Housing） | 35% | 同等区域租金/单价 |
| 餐饮（Food） | 25% | 大米(kg)、猪肉(kg)、社交餐饮 |
| 交通（Trans） | 10% | 公交票价、出租车起步价、92号汽油 |
| 医疗（Health） | 10% | 挂号费、感冒药单价 |
| 教育（Edu） | 10% | 公办幼儿园/小学/初中平均学费 |
| 服务（Service） | 10% | 剪发、家政时薪、电影票价 |

计算公式：
- 基准：上海COLI(SH)=100
COLI_X = Σ（P_X,i / P_SH,i × w_i）×100
- P_X,i：城市X维度i平均价格
- P_SH,i：上海维度i平均价格
- w_i：维度权重

## 交互流程
### 时空等值转换器
1. **输入阶段**
   - 首页默认显示“时空转换”tab
   - 输入原始金额（1-99,999,999）
   - 下拉选起始年份（1980-2025，默认1990）
   - 选择城市（首期支持省会/直辖市，二期扩至地级市）
2. **触发阶段**
   - 点击“开启时光机”，按钮loading，播放0.5秒时光穿梭动效
3. **计算阶段**
   - 并行调取CPI/M2静态表、历史实物价格向量库
4. **展示阶段**
   - 顶部：醒目展示转换后现值
   - 中部：实物卡片（默认“吃”，可切换“住”“行”）
   - 底部：专业逻辑入口，查看三位一体公式与占比

### 财富漂流瓶
1. **前置条件**：切换至“财富漂流”页
2. **用户输入**：所在城市、月收入、总资产（可选）、目标城市
3. **触发与计算**
   - 点击“开始漂流”，按钮loading，播放0.5秒漂流瓶动效
   - 调取两地COLI，计算比值与等效购买力
   - 示例：月入20000元，目标COLI=52，所在COLI=100 → 等效≈38461元
4. **身份标签判定**
   - 公式：Wealth_Ratio = 等效金额 / 目标城市中位数收入
   - 规则：
     - >5：县城土豪→吃饭自由
     - 2<≤5：体面名流→高薪阶层
     - 0.8<≤1.2：平替生活→打工味不变
     - ≤0.8：生存挑战→压力极大
5. **可视化输出**
   - 购买力平移图：左右对比两地生活水平
   - 等值数字+实物对照表（RAG检索物价）
